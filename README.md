## Самостоятельная работа №4
### Манипулирование строками. Напишите программу на Python, которая принимает предложение (на английском) в качестве входных данных от пользователя. Выполните следующие операции и отобразите результаты: 1)Выведите длину предложения. 2)Переведите предложение в нижний регистр. 3)Подсчитайте количество гласных (a, e, i, o, u) в предложении. 4)Замените все слова "ugly" на "beauty". 5)Проверьте, начинается ли предложение с "The" и заканчивается ли на "end". Проверьте работу программы минимум на 3 предложениях, чтобы охватить проверку всех поставленных условий.

### Результат.
```python
vowels = ["a","e","i","o","u","A","E","I","O","U"]
counter = 0

string = input("Enter your sentence: ")

# 1. Длина предложения
print(f"Lenght of sentence is {len(string)}")

# 2. Предложение в нижнем регистре
print(f"Your sentence in lower case: {string.lower()}")

# 3. Кол-во глассных
for i in range(len(string)):
    if string[i] in vowels:
        counter += 1
print(f"Amount of vowels is {counter}")
# Альтернативый способ: print(string.count("a") + string.count("e") + string.count("i") + string.count("o") + string.count("u"))

# 4. Замена ugly на beauty
print("Replace ugly with beauty: " + string.replace("ugly","beauty"))

# 5. Проверка начинается ли с The и заканчивается ли на end
if string[:3] == "The" and string[(len(string)-3):(len(string))] == "end":
    print("Your sentence starts with \"The\" and ends with \"end\"")
else:
    print("Your sentence isn't start with \"The\" and end with \"end\"")
```

![Меню](https://github.com/qwedani/Software_Engineering/blob/Тема-3/pic/Ex4.png)

## Выводы
- В начале программы объявляются переменные: четные буквы, счетчик (понадобится для подсчета четных букв) и строка, которую должен будет ввести пользователь
  1. `len(string)` - считает длину предложения
  2. `string.lower()` - переводит предложение в нижний регистр
  3. С помощью прохода по всему предложению `for i in range(len(string))` и сравнения каждой буквы с заранее заданым массивом `if string[i] in vowels:` считается кол-во гласных `counter += 1`
  4. `string.replace("ugly","beauty")` - заменяет ugly на beauty
  5. `if string[:3] == "The" and string[(len(string)-3):(len(string))] == "end":` - проверяется равняется ли первые три символа слову "The" и последние три символа слову "end" (наверное стоило бы проверять последние 4 символа, чтобы учесть пробел и избежать учет таких слов как friend, girlfriend, send, spend и так далее)
  Так же есть другой способ проверить с чего начинается предложение и чем заканчивается: методы `string.startwith()` и `string.endwith()`. Об их существовании узнал после завершения задания.

## Самостоятельная работа №5
### Создать программу из кусочков кода.

### Результат.
```python
string = "hello"
counter = 0
values = [0,2,4,6,8,10]

while "world" not in string:
    memory = string
    if counter in values:
        string = string + " world"
    print(string)
    if counter < 10:
        string = memory
    counter += 1
```

![Меню](https://github.com/qwedani/Software_Engineering/blob/Тема-3/pic/Ex5.png)

## Выводы
- В начале программы задаются переменные: строка "hello", счетчик, массив [0,2,4,6,8,10]
- `while "world" not in string:
    memory = string
    if counter in values:
        string = string + " world"
    print(string)
    if counter < 10:
        string = memory
    counter += 1` - производит проверку есть ли в строке слово "world", если есть, то переменной memory присваивается значение строки string, после чего начинается следующий цикл "if" проверяющий состоит ли счетчик в заданном массиве, если да, то прибавляет к строке слово "world" (либо не присваивает, если значение счетчика не входит в массив), после чего строка выводится. Дальше проверяется достиг ли счетчик 10, если нет то переменная memory переписывается, после чего к счетчику прибавляется еденица. И так пока значение переменной счетчика не будет равно 10
